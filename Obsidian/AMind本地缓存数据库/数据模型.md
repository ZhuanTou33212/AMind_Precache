# 数据模型

## 1. SQLite 数据库

### 数据库文件
- 路径: `data/cache.db`
- 驱动: `modernc.org/sqlite` (纯 Go，无需 CGO)
- 写模式: WAL (Write-Ahead Logging)

### images 表

```sql
CREATE TABLE IF NOT EXISTS images (
    hash         TEXT PRIMARY KEY,   -- URL SHA1
    url          TEXT,               -- OSS 原始 URL
    prompt_id    TEXT,               -- AMiner 题目 UUID
    question_num INTEGER,            -- 全局题号 (1-2000)
    reason       TEXT,               -- auto/prefetch/manual
    cached_at    INTEGER,            -- Unix 毫秒时间戳
    file_size    INTEGER             -- 字节数
);
```

### 图片文件命名

```
data/images/{sha1}.img

sha1 = SHA1(原始 URL)
扩展名: .img
```

## 2. 配置文件

`data/config.json`:

```json
{
  "token": "eyJhbGciOiJIUzUxMiIsInR5cCI6IkpXVCJ9...",
  "taskId": "181903",
  "startDate": "1781744400",
  "cacheSize": 50
}
```

| 字段 | 来源 | 说明 |
|------|------|------|
| token | 浏览器 localStorage `Access-Token` | JWT，约 2 小时有效 |
| taskId | URL `label_page_feed/{taskId}` | 标注任务 ID |
| startDate | URL `?start={startDate}` | Unix 时间戳 |
| cacheSize | 用户配置 | 兼容字段；实际推荐使用扩展配置文件 |

扩展缓存数量配置文件 `extension/settings.json`:

```json
{
  "cacheSize": 200
}
```

`extension/settings.json` 优先级高于 `data/config.json`，范围 1-200。

## 3. AMiner API 响应格式

### 题目列表

`GET /api/v1/annotations/annot/prompts/task/{taskId}/date/{startDate}/v2?page={page}`

```json
{
  "prompts": [{
    "id": 24442965,
    "prompt_id": "01480083-357a-4afc-9d32-56c00c85b055",
    "state": 1,
    "Prompt": " "
  }],
  "page": 1,
  "total_page": 100,
  "page_size": 20,
  "indicator": {
    "total_cnt": 2000,
    "unlabeled_page": 85,
    "annot_cnt": 1687
  }
}
```

**分页公式**: 题号 = (page-1)×20 + 列表中位置

**state 含义**: 0=未标注, 1=已标注

### 题目详情

`GET /api/v1/bench/questions/{prompt_id}?uid=`

```json
{
  "id": "01480083-...",
  "prompt": " ",
  "payload": {
    "internal_id": "cg_20260611_xxx",
    "est_quality": 94.1
  },
  "responses": [{
    "reply": "![img](https://mm-group-image.oss-cn-beijing.aliyuncs.com/xxx?OSSAccessKeyId=...&Expires=...&Signature=...)"
  }]
}
```

### 图片 URL 解析

从 `reply` 字段的正则提取: `/https:\/\/mm-group-image\.oss-cn-beijing\.aliyuncs\.com\/[^\s"'<>\\)]+/g`

## 5. 缓存策略

- 容量: 默认 **50 张**，可通过 `extension/settings.json` 的 `cacheSize` 调整
- 触发: 标注提交后自动检查
- 超出: 清除全部旧缓存，重新从当前题拉取 `cacheSize` 张
- 存储: 图片存磁盘，元数据存 SQLite
