# 编译说明

## 桌面应用

### 语言与工具链

- **语言**: Go (Golang) 1.21+
- **编译器**: `go build`

### 依赖

```
modernc.org/sqlite v1.29.5  # 纯 Go SQLite (无 CGO)
```

其他均为 Go 标准库，零外部依赖。

### 编译命令

```bash
# 1. 进入源码目录
cd aminer-desktop/

# 2. 下载依赖
GOPROXY=https://goproxy.cn,direct go mod tidy

# 3. 编译
go build -ldflags="-s -w" -o aminer-desktop.exe .

# -ldflags="-s -w" 去除调试信息，压缩体积
```

### 源码文件

| 文件 | 职责 |
|------|------|
| `main.go` | 入口、HTTP 路由、API 代理、窗口置顶 |
| `store/store.go` | SQLite 数据库 + 磁盘文件 CRUD |
| `ui/index.html` | 嵌入式 Web 管理界面 |
| `go.mod` | 模块声明 + 依赖 |

### 平台要求

- **OS**: Windows（`pinTopmost` 使用 `user32.dll` 系统调用）
- **端口**: 9800 (可通过环境变量 `AMINER_PORT` 修改)
- **存储**: `data/` 子目录（与 exe 同目录）

### 跨平台适配

移除 `pinTopmost()` 函数及相关 `syscall` 调用后可编译 Linux/macOS 版本。

## 浏览器扩展

### 语言

JavaScript ES6+，零依赖，纯浏览器 API。

### 文件

| 文件 | 用途 |
|------|------|
| `manifest.json` | Chrome MV3 清单 |
| `background.js` | Service Worker |
| `monitor.js` | 内容脚本（隔离世界） |
| `monitor-main.js` | 内容脚本（主世界） |

### 加载

1. Chrome/Edge → `chrome://extensions/`
2. 开启「开发人员模式」
3. 「加载解压缩的扩展」→ 选择 `extension/` 目录
4. 无需编译，直接加载源码

### manifest.json 关键配置

```json
{
  "manifest_version": 3,
  "host_permissions": ["http://127.0.0.1:9800/*"],
  "background": { "service_worker": "background.js" },
  "content_scripts": [
    {
      "matches": ["...label_page_feed/*"],
      "js": ["monitor.js"],
      "run_at": "document_start"
    },
    {
      "matches": ["...label_page_feed/*"],
      "js": ["monitor-main.js"],
      "run_at": "document_start",
      "world": "MAIN"
    }
  ]
}
```

- `host_permissions` 允许 Service Worker 访问 localhost
- `world: "MAIN"` 让 monitor-main.js 运行在页面主世界（才能拦截页面 XHR）
