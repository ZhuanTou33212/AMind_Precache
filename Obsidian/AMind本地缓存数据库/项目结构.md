# 项目结构

## 目录树

```
AMiner预缓存系统/
├── aminer-desktop.exe          # Go 编译的桌面服务端
├── extension/                  # Chrome/Edge 浏览器扩展
│   ├── manifest.json           # 扩展清单
│   ├── background.js           # Service Worker (核心引擎)
│   ├── monitor.js              # 内容脚本-隔离世界
│   └── monitor-main.js         # 内容脚本-主世界
├── data/                       # 运行时数据
│   ├── config.json             # 用户配置
│   ├── cache.db                # SQLite 数据库
│   └── images/                 # 图片文件 {sha1}.img
└── Obsidian/                   # 本文档
```

## 进程模型

```
┌─ aminer-desktop.exe ──────────────────┐
│  HTTP Server (127.0.0.1:9800)         │
│  ├─ /api/config    配置读写            │
│  ├─ /api/prompts   AMiner API 代理    │
│  ├─ /api/question  题目详情代理        │
│  ├─ /api/cache     图片下载缓存        │
│  ├─ /api/image/    图片文件服务        │
│  ├─ /api/images    缓存列表 CRUD       │
│  ├─ /api/monitor   页面状态接收        │
│  ├─ /api/pin       窗口置顶           │
│  ├─ SQLite 数据库                     │
│  └─ 磁盘图片文件                       │
└────────────────────────────────────────┘
         ↕ HTTP (本地, 无CORS)
┌─ 浏览器扩展 ───────────────────────────┐
│  background.js (Service Worker)       │
│  ├─ 接收 monitor.js 消息               │
│  ├─ 调用 exe API 代理 AMiner API       │
│  ├─ 可配置缓存生命周期管理              │
│  └─ 推送缓存列表给 monitor.js           │
│                                        │
│  monitor.js (隔离世界 content script)   │
│  ├─ 接收 background 的缓存列表          │
│  ├─ 替换页面 img.src OSS→localhost     │
│  └─ 转发主世界事件给 background         │
│                                        │
│  monitor-main.js (主世界 content script)│
│  ├─ 拦截 XMLHttpRequest 检测提交       │
│  └─ 读取页面 "第X题/Y" 文字            │
└────────────────────────────────────────┘
```

## 通信链路

| 方向 | 机制 | 说明 |
|------|------|------|
| monitor-main.js → monitor.js | `CustomEvent('aminer-monitor:event')` | 主世界→隔离世界 |
| monitor.js → background.js | `chrome.runtime.connect` port | 持久连接 |
| background.js → monitor.js | port.postMessage | 推送缓存列表 |
| background.js → exe | `fetch('http://127.0.0.1:9800/...')` | HTTP API |
| exe → AMiner | `http.Get(url)` + Bearer Token | API 代理 |

## 关键文件行数

| 文件 | 行数 | 职责 |
|------|------|------|
| `background.js` | ~170 | 缓存引擎、API 代理、跨页逻辑 |
| `monitor.js` | ~140 | 图片替换、缓存列表同步 |
| `monitor-main.js` | ~85 | 提交检测、页面信息 |
| `main.go` | ~260 | HTTP 服务、API 代理、窗口管理 |
| `store/store.go` | ~160 | SQLite + 文件存储 |
| `ui/index.html` | ~240 | Web 管理界面 |
