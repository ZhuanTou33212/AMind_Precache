# AMiner 图片预缓存系统

## 一句话描述

为 AMiner 标注平台提供**本地图片预缓存**，将云端 OSS 图片提前下载到本地磁盘，标注页面从本机加载，实现图片秒开。

## 项目组成

| 组件 | 语言 | 位置 | 说明 |
|------|------|------|------|
| 桌面服务端 | Go | `aminer-desktop.exe` | HTTP 服务器 + SQLite + 图片存储 |
| 浏览器扩展 | JavaScript | `extension/` | Service Worker + 内容脚本 |

## 关键数据流

```
标注页面 → 检测提交 → Service Worker → AMiner API → 下载图片 → 本地磁盘
                                                         ↓
                                           替换页面 img.src → 图片秒开
```

## 系统边界

- **不修改** 标注页面 DOM 结构
- **不拦截** 页面网络请求
- **不影响** Hotkeys 等其他扩展
- **不依赖** 外部数据库或云服务
