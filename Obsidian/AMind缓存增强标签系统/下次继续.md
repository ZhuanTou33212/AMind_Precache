## 推荐第一步

编辑：

```text
extension/monitor-quicklabel.js
src-aminer-desktop/extension/monitor-quicklabel.js
```

新增调试对象：

```js
window.__aminerQuickLabel = {
  scanOptions,
  applyLabels,
  currentLabels
}
```

`scanOptions()` 要返回当前 AMiner 页面识别到的 radio、checkbox、button、label 文本。

## 推荐实现顺序

1. 扫描页面选项。
2. 用标签 `label + aliases` 匹配网页文案。
3. 自动点击选项。
4. 做 `Alt+Enter` 单题提交。
5. 做 `Esc` 暂停。
6. 做 `Alt+B` 连续 5 题批量提交。

## 风险提醒

- 批量提交前必须校验当前网页题号等于本地缓存 `questionNum`。
- 找不到网页选项时必须停止，不允许跳过。
- 第一版继续用页面按钮/快捷键提交，不要直接 POST AMiner 云端提交接口。

## 最新进度：预标注窗口

已新增 EXE 内的预标注窗口，不再依赖复杂 JSON 编码栏完成图片标注。

当前机制：

```text
当前题号 -> 未标注缓存层取图 -> 预标注窗口 -> 保存 -> 已标注数据缓存层 -> 下一张未标注图
```

