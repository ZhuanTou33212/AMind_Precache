# 项目结构

```
AMiner预缓存系统/
│
├── aminer-desktop.exe          # Go 编译的桌面服务端（双击启动）
│
├── extension/                  # Chrome/Edge 浏览器扩展
│   ├── manifest.json           # 扩展清单（权限、内容脚本配置）
│   ├── background.js           # Service Worker — 核心预缓存引擎
│   ├── monitor.js              # 内容脚本（隔离世界）— 图片替换 + 监控桥接
│   └── monitor-main.js         # 内容脚本（主世界）— 提交检测 + 页面信息
│
├── data/                       # 运行时数据（自动生成）
│   ├── config.json             # 用户配置（Token 等）
│   ├── cache.db                # SQLite 数据库（图片元数据）
│   └── images/                 # 图片文件存储
│       └── {sha1}.img          # 以 URL 的 SHA1 哈希命名
│
├── Obsidian/                   # 文档
│   ├── 项目结构.md
│   ├── 数据模型.md
│   ├── 编译说明.md
│   ├── 使用说明.md
│   └── 工作原理.md
│
└── 使用说明.html               # 用户指南（浏览器打开）
```

## 进程模型

```
┌─ aminer-desktop.exe ────────────────┐
│  HTTP Server (:9800)                 │
│  ├─ /api/config     配置读写         │
│  ├─ /api/prompts    AMiner API 代理  │
│  ├─ /api/question   题目详情代理     │
│  ├─ /api/cache      图片下载缓存     │
│  ├─ /api/image/{h}  图片文件服务     │
│  ├─ /api/images     缓存列表 CRUD    │
│  ├─ /api/monitor    页面状态接收     │
│  └─ /api/pin        窗口置顶         │
│  SQLite DB                           │
│  磁盘文件存储                         │
└──────────────────────────────────────┘
         ↕ HTTP (localhost, 无CORS)
┌─ 浏览器扩展 ─────────────────────────┐
│  background.js (Service Worker)      │
│  ├─ 接收页面事件                     │
│  ├─ 调用本地 API 代理 AMiner API     │
│  ├─ 管理 50 张缓存生命周期           │
│  └─ 推送缓存列表给内容脚本            │
│  monitor.js (隔离世界)               │
│  ├─ 接收缓存列表                     │
│  ├─ 替换页面 OSS URL → localhost     │
│  └─ 转发主世界事件                   │
│  monitor-main.js (主世界)            │
│  ├─ 拦截 XHR/fetch 检测提交          │
│  └─ 读取页面"第X题/Y"文字            │
└──────────────────────────────────────┘
```
