# 使用说明

## 安装

### 1. 启动服务端

双击 `aminer-desktop.exe`，保持黑色终端窗口运行。

### 2. 加载扩展

```
edge://extensions/ → 开发人员模式 → 加载解压缩的扩展
→ 选择 extension/ 文件夹
```

### 3. 获取 Token

打开标注页面 → F12 → Console → 输入 `允许粘贴` 回车 → 粘贴：

```js
var _s=XMLHttpRequest.prototype.setRequestHeader;
XMLHttpRequest.prototype.setRequestHeader=function(k,v){
  if(k.toLowerCase()==='authorization') console.log('TOKEN:',v.replace('Bearer ',''));
  return _s.apply(this,arguments);
}
```

点任意数字键(1-5)提交标注 → Console 出现 `TOKEN: eyJhbGci...` → 复制。

### 4. 配置 Token

浏览器打开 `http://127.0.0.1:9800/` → 粘贴 Token → 点「💾 保存」。

---

## 日常使用

**正常标注即可，无需额外操作。**

系统自动完成：
- 检测当前题目
- 从 API 获取后续 50 张未标注题
- 下载图片到本地磁盘
- 替换页面 OSS URL 为 localhost

---

## 管理界面

打开 `http://127.0.0.1:9800/`：
- 📌 置顶 — 窗口钉在最上层
- 💾 保存 — 保存 Token 配置
- 📋 加载题目 — 查看题目列表和状态
- 🚀 预取 — 手动预取图片
- 🗑 清空 — 删除全部缓存

---

## Token 过期

Token 约 2 小时过期，现象：图片不再缓存、Console 出现 401。

按「获取 Token」步骤重新获取即可。

---

## 故障排查

| 现象 | 检查 |
|------|------|
| 图片没替换 | exe 是否运行？扩展是否加载？ |
| 401 错误 | Token 是否过期？ |
| CORS 错误 | 扩展是否刷新？background.js 是否最新？ |
| exe 无法启动 | 是否已有实例在运行？端口 9800 是否被占用？ |
