# 数据模型

## 1. 图片记录 (SQLite — `cache.db`)

### images 表

| 字段 | 类型 | 说明 |
|------|------|------|
| hash | TEXT (主键) | 图片 URL 的 SHA1 哈希值 |
| url | TEXT | OSS 原始图片 URL |
| prompt_id | TEXT | 关联的题目 UUID |
| question_num | INTEGER | 全局题号 (1-2000) |
| reason | TEXT | 缓存原因 (`auto` / `prefetch` / `manual`) |
| cached_at | INTEGER | 缓存时间 (Unix 毫秒时间戳) |
| file_size | INTEGER | 图片文件大小 (字节) |

### 图片文件

| 路径 | 说明 |
|------|------|
| `data/images/{hash}.img` | 以 hash 为文件名的 JPEG/PNG 图片 |

### 命名规则

```
hash = SHA1(原始URL)
文件名 = hash + ".img"
```

## 3. AMiner API 数据结构

### 题目列表 (`/api/v1/annotations/annot/prompts/task/{id}/date/{ts}/v2?page=N`)

```json
{
  "prompts": [
    {
      "id": 24442965,
      "prompt_id": "01480083-357a-4afc-9d32-56c00c85b055",
      "state": 1,         // 1=已标注, 0=未标注
      "Prompt": " "
    }
  ],
  "page": 1,
  "total_page": 100,
  "page_size": 20,
  "indicator": {
    "total_cnt": 2000,
    "unlabeled_page": 85,
    "annot_cnt": 1687
  }
}
```

### 题目详情 (`/api/v1/bench/questions/{prompt_id}?uid=`)

```json
{
  "id": "01480083-357a-4afc-9d32-56c00c85b055",
  "prompt": " ",
  "payload": {
    "internal_id": "cg_20260611_xxx",
    "est_quality": 94.1
  },
  "responses": [
    {
      "reply": "![img](https://mm-group-image.oss-cn-beijing.aliyuncs.com/xxx)"
    }
  ]
}
```

### 图片 URL 格式

```
https://mm-group-image.oss-cn-beijing.aliyuncs.com/{hash}?OSSAccessKeyId=...&Expires=...&Signature=...
```

## 4. 扩展内部消息格式

### 内容脚本 → Service Worker

```json
{
  "question": 1973,     // 当前题号
  "total": 2000         // 总题数
}
```

```json
{
  "type": "submission",  // 标注提交事件
  "question": 1973
}
```

### Service Worker → 内容脚本

```json
{
  "type": "cache-list",
  "images": [
    {
      "hash": "1e272378...",
      "url": "https://mm-group-image...",
      "questionNum": 1973,
      "promptId": "01480083-..."
    }
  ]
}
```

### 主世界 → 隔离世界

通过 `window.dispatchEvent(new CustomEvent('aminer-monitor:event', {detail}))` 传递。
