# 编译说明

## 桌面应用 (`aminer-desktop.exe`)

### 语言
Go (Golang)

### 依赖
- `modernc.org/sqlite` — 纯 Go SQLite 驱动（无 CGO）
- Go 标准库 (`net/http`, `encoding/json`, `database/sql`, `syscall` 等)

### 编译命令

```bash
# 安装 Go: https://go.dev/dl/

# 下载依赖
go mod tidy

# 编译（-ldflags="-s -w" 压缩体积）
go build -ldflags="-s -w" -o aminer-desktop.exe .
```

### 源码文件

| 文件 | 用途 |
|------|------|
| `main.go` | HTTP 服务器、API 代理、窗口置顶 |
| `store/store.go` | SQLite 数据库 + 磁盘文件存储 |
| `go.mod` | 模块定义 + 依赖 |
| `ui/index.html` | 嵌入式 Web 管理界面 |

### 编译环境要求

- Go ≥ 1.21
- Windows (syscall 调用了 Windows API)
- 网络访问（下载依赖）

### 跨平台说明

当前仅支持 Windows（使用了 `user32.dll` 窗口置顶 API）。如需 macOS/Linux，需移除 `pinTopmost()` 函数。

## 浏览器扩展

### 语言
JavaScript (ES6+)

### 文件

| 文件 | 行数 | 说明 |
|------|------|------|
| `background.js` | ~160 | Service Worker，核心缓存逻辑 |
| `monitor.js` | ~135 | 内容脚本（隔离世界），图片替换 |
| `monitor-main.js` | ~85 | 内容脚本（主世界），提交检测 |
| `manifest.json` | ~20 | Chrome 扩展清单 |

### 零依赖

扩展不依赖任何外部库，纯浏览器 API。

### 编译

无需编译。Chrome/Edge 直接加载源码目录。
